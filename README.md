# Tripleten web_project_around_express
Around the U.S. – Express API

API REST desarrollada con Node.js y Express para el proyecto Around the U.S..
La aplicación proporciona endpoints para usuarios y tarjetas, leyendo los datos desde archivos JSON mediante el módulo fs.

Tecnologías

-Node.js
-Express
-ESLint
-fs (File System)
-path
